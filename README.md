## 1. top 명령어

### 📌 개요
`top`은 시스템의 **실시간 프로세스 상태**를 모니터링하는 명령어입니다. CPU 사용률, 메모리 사용량, 실행 중인 프로세스 정보를 동적으로 보여줍니다.

**각 항목의 의미:**
- **load average**: 1분, 5분, 15분 평균 시스템 부하
- **Tasks**: 전체 프로세스 수와 상태별 분류
- **%Cpu(s)**: CPU 사용률 (us: 사용자, sy: 시스템, id: 유휴)
- **MiB Mem**: 메모리 사용 현황
- **MiB Swap**: 스왑 메모리 사용 현황


**컬럼 설명:**
- **PID**: 프로세스 ID
- **USER**: 프로세스 소유자
- **PR**: 우선순위 (Priority)
- **NI**: Nice 값 (-20 ~ 19)
- **VIRT**: 가상 메모리 사용량
- **RES**: 실제 메모리 사용량
- **SHR**: 공유 메모리
- **S**: 프로세스 상태 (R: 실행, S: 대기, Z: 좀비)
- **%CPU**: CPU 사용률
- **%MEM**: 메모리 사용률
- **TIME+**: 누적 CPU 사용 시간
- **COMMAND**: 실행 명령어

### ⌨️ 주요 옵션 및 단축키

| 옵션/키 | 설명 |
|---------|------|
| `top -d 5` | 5초마다 화면 갱신 |
| `top -u username` | 특정 사용자의 프로세스만 표시 |
| `top -p PID` | 특정 PID만 모니터링 |
| `q` | top 종료 |
| `k` | 프로세스 종료 (kill) |
| `M` | 메모리 사용량 순으로 정렬 |
| `P` | CPU 사용량 순으로 정렬 |
| `h` | 도움말 표시 |
| `1` | CPU 코어별로 표시 |
